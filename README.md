# 📚 TPI-OE-Caramuti-Giordano

## Automatización del proceso de gestión de turnos para Eudermy Estética

Trabajo Práctico Integrador desarrollado para la materia **Organización Empresarial**, cuyo objetivo fue analizar un proceso de negocio real y proponer una solución de automatización mediante el modelado de procesos y el uso de un chatbot de Telegram.

---

## 📖 Descripción

El proyecto se centra en la optimización del proceso de gestión de turnos de **Eudermy Estética**.

A partir del análisis del proceso actual (**AS-IS**) se identificaron oportunidades de mejora y se diseñó un nuevo proceso (**TO-BE**) utilizando la notación **BPMN 2.0**, incorporando un chatbot de Telegram como herramienta de automatización para la atención al cliente.

## 🎯 Objetivos

- Analizar el proceso actual de gestión de turnos.
- Detectar ineficiencias y oportunidades de mejora.
- Modelar el proceso AS-IS.
- Diseñar el proceso TO-BE.
- Automatizar la gestión de turnos mediante un chatbot.
- Mejorar la experiencia del cliente y optimizar el trabajo administrativo.

---

## ⚙️ Funcionalidades propuestas

La solución diseñada permite:

- 📅 Solicitar turnos.
- 🔍 Consultar reservas.
- ❌ Cancelar turnos.
- ✔️ Verificar disponibilidad de horarios.
- 🤖 Confirmar reservas automáticamente.
